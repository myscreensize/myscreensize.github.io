UNDER THE HOOD

How It Works

We use native browser APIs — no tracking, no server calls, no data collection.

01

Browser APIs

We query window.screen, window.devicePixelRatio, and navigator APIs directly — all data stays in your browser.

02

Live Calculation

A ResizeObserver and resize event listener keeps every metric synced with your actual browser viewport in real time.

03

Zero Storage

Nothing is sent to any server. All detection runs purely client-side using JavaScript. Your data never leaves your device.
